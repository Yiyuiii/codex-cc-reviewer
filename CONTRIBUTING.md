# Contributing

This project intentionally stays narrow.

Accepted contributions:

- better review prompts
- safer defaults
- better Codex install support
- better Claude CLI parsing
- tests
- docs
- support for new Claude Code flags

Out of scope for now:

- full bidirectional bridges
- autonomous file editing
- web dashboards
- cloud-hosted review services

Before opening a pull request, run:

```bash
npm run typecheck
npm test
npm run build
```
